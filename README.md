# Flora Vision 🌿

A responsive frontend landing page for a plant e-commerce platform built as part of a frontend development assignment.

## 📌 Project Overview

Flora Vision is a modern and responsive product listing website focused on indoor plants and greenery products. The project recreates the provided Figma design with attention to layout structure, responsiveness, typography, spacing, and interactive UI elements.

The website includes:

- Responsive navigation bar
- Hero section with CTA buttons
- Trending plants showcase
- Top selling products section
- Customer testimonials
- Featured O₂ plants section
- Footer with quick links and newsletter form
